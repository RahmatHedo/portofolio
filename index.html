About me

Halo, saya Rahmat Hedo Alfikri, seorang beginner front end developer yang sangat bersemangat untuk belajar dan berkembang dalam dunia pengembangan web. Saya tertarik dengan teknologi web dan berfokus untuk menciptakan pengalaman pengguna yang responsif, interaktif, dan menarik.

Saat ini, saya sedang memperdalam keterampilan saya dalam HTML, CSS, dan JavaScript, serta membangun aplikasi web dengan menggunakan framework seperti ini. Saya memiliki ketertarikan khusus terhadap desain antarmuka pengguna (UI) dan pengalaman pengguna (UX), serta terus memperbarui pengetahuan saya tentang tren terbaru dalam pengembangan front-end.

Sebagai seorang pemula, saya tidak hanya berfokus pada penguasaan teknologi, tetapi juga pentingnya kolaborasi tim, dokumentasi yang jelas, dan pengembangan berkelanjutan. Saya sangat antusias untuk terus mengasah keterampilan saya dan siap untuk mengambil tantangan baru di dunia pengembangan web.
